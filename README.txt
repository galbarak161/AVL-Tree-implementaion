Implementaion of AVL tree in C using struct without parent pointer.
The application includes the following basic functions:

new_avl_node (int k)
avl_insert (AVLNodePtr tnode, int k)
avl_delete (AVLNodePtr tnode, int k)
avl_search (AVLNodePtr tnode, int k)
delete_avl_tree (root AVLNodePtr)

And more!
